## 🚀 Sobre el Repositorio

Este es el repositorio central del grupo **Escuchify** para la cátedra de **Ingeniería de Software 3 / Actualidad Informática**. Aquí se recopilarán todos los trabajos prácticos, informes técnicos y desarrollos realizados durante el ciclo lectivo 2026.

El objetivo de este espacio es mantener un historial versionado de nuestras actividades académicas, asegurando la trazabilidad y organización de la documentación producida por el equipo.
